# Basic CI/CD Concepts with Jenkins & Github 

Jenkinsfile — Basic CI Pipeline pipeline 
{ 
    agent any stages { 
        stage('Build') { 
            steps { echo 'Building project' }
        } 
        stage('Test') { 
            steps { echo 'Running tests' } 
        } 
        stage('Deploy') {
            steps { echo 'Deploying application' } 
        } 
    } 
}

GitLab CI/CD — Basic Pipeline 

stages:
    - build
    - test
    - deploy

build: 
    stage: build 
    script: 
        - echo "Building project"

test:
    stage: test 
    script: 
        - echo "Running tests" 

deploy: 
    stage: deploy 
    script: 
        - echo "Deploying application" 


Ansible - Install and Configure Nginx

- name: Install and configure Nginx 
  hosts: all 
  become: yes 
  tasks: 
    - name: install nginx 
      yum: 
        name: nginx 
        state: present

    - name: Configure nginx 
      template: 
        src: nginx.conf.js2 
        dest: /etc/nginx/nginx.conf

    - name: Start nginx service 
      service: 
        name: nginx 
        state: started 
        enabled: yes
